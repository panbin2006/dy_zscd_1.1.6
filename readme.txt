1.实现的功能：
	a)用户登陆;

	b)计划单浏览，新增、修改、删除;

	c)送货单浏览，新增、修改、删除;

	d)材料消耗查询;

	e)材料库存;

	f)车辆查询;

	g)退转料查询;

2.页面设计：
	a)用户登陆页面：部门、用户、密码.

	b)软件主页：
		1）.调度中心：管理调配生产与设备管理

		2）.原材料管理：原材料综合分析

		3）.配合比追踪：管理混凝土生产配合比

		4）.底栏:首页、工作、通讯录、我的

	c)计划单查询页面：
		1）.按施工状态查询：全部、开工、完工、等待
		2）.列表显示：
			左边：计划单号、工程名称、部位、计划开盘
			右边：施工状态、坍落度、强度等级、配方号
		3）任务单详情页面：计划单号、施工状态、工程名称、施工部位、浇注方式
				坍落度、砼标号、施工单位、计划开盘时间、备注
	
	d)送货单查询页面
		1）按生产线查询:全部、1线、2线
		2）列表显示
			左边：计划单号、工程名称、部位、砼标号、车号
			右边：本车方量、累计方量、累计车次
		
	e)材料消耗查询页面
		上部显示柱形图
		下部显示图表：材料名称规格、配比用量（kg）、施工用量、误差值、误差率

	f)材料库存页面
	
	g)车辆查询：
	车辆状态查询：出厂、待命、异常
	列表显示：左边显示 车牌号、司机、刷卡时间     右边显示 车辆状态、容量

	h)调度中心  
	    1）计划量、生产量（饼图）：已开盘、未开盘、今日任务单（跳转任务单模块）、车辆查询（跳转车辆查询模块）
            2）本月退转方量图表
	I)退转明细
	    1）按强度等级分组显示：
	      分组头：强度  
	      分组数据：退回总量、退回已签、退回未签、转出方量、转入方量、转出方量、转入方量、作废方量
		
	      2）按工程名称分组显示：
	      分组头：工程名称
	      分组数据：退回总量、退回已签、退回未签、转出方量、转入方量、转出方量、转入方量、作废方量
